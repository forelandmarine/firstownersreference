## Template 1 — print copy outreach (lawyers, naval architects, family-office advisers)

For the 20 named contacts. One-to-one, signed by Jack or Dan. No CC list, no link batches, no autoresponder.

Subject: A first edition for your reference shelf

```
[Recipient first name],

Foreland Marine is publishing the first edition of The First Owner's
Reference in September. It is an annual editorial publication for
first-time superyacht buyers, set across nine chapters covering the
structural, financial, and operational dimensions of acquisition,
new build, refit, and operation. Independent. No advertising.
Contributor-led.

I have set aside a print copy in your name. The print run is 500,
hand-numbered, and we are placing copies with a small number of
practitioners we would value reading the publication carefully and
telling us where it is wrong.

If you would like a copy, just reply with the address you would like
it sent to, and a name for the attribution if it should be anything
other than yours. We will post in early October.

Editorial correspondence and corrections, as always, reach us at
editors@firstownersreference.com.

With thanks,
Jack
```

**Personalisation pass before sending.** One sentence at the top tying the recipient's recent published work, recent transaction, or known specialism to the chapter that most fits them. One sentence at the bottom suggesting which chapter you would value their read on most. The recipient should feel the note was written for them and not adapted from a template.

**Recipients (20).** Pull from the chapter 09 introduction list and Foreland's working network.
- 5 maritime / shipbuilding lawyers across Hill Dickinson, WFW, Stephenson Harwood, HFW, NRF
- 5 naval architects covering Vitruvius, Reymond Langton, Espen Oeino, Bannenberg & Rowell, Newcruise
- 5 family-office advisers across London / Geneva / Monaco
- 5 independent owner's-rep peers across A2B, ULTIMAR, Mariner Technical Services, Praxis Marine, Saor Alba

**Cadence.** All twenty go out in a single week, between June 1 and June 7, so the print copies arrive at the same time. Replies are tracked manually in a private spreadsheet; no CRM.

With thanks,
Jack

## Notes for both editors

- Keep templates as sketches. The personal note is the work; the template is the structural memory.
- Track all outgoing outreach in a private spreadsheet (Google Sheet or Notion table). Columns: recipient, role, firm, route, sent date, replied y/n, outcome, follow-up date.
- Two-week follow-up rule: if no reply within two weeks, one short follow-up. If no reply within four weeks total, close the thread cleanly and document.
- Both editors sign off on outreach before sending. Single editor signing is fine; both editors should know who has been contacted, when, and why.
